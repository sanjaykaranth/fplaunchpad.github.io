NPTEL Course: Functional Programming with OCaml

KC Sivaramakrishnan is offering Functional Programming with OCaml on NPTEL in the July 2026 semester. Enrollment is open now, free for all.

The course comes with an interactive book where you can run OCaml directly in your browser. The goal: make 0-to-OCaml take 0 steps, including using OCaml's platform tools: no installation, no setup, just open and start coding.
Technical note Your OCaml program runs in a bytecode interpreter, on an emulated x86 CPU, JIT-compiled to Wasm, running in your browser, JIT-compiled to run on your local machine.

The first 8 modules are stable; the rest is being built in the open.

PaPoC 2026: Keynote and Paper

KC Sivaramakrishnan gave an invited keynote, "From Convergence to Confidence: Push-Button Verification for Replicated Data Types", at PaPoC 2026 (co-located with EuroSys, Edinburgh). The talk argues that convergence alone is not enough to verify replicated data types, and introduces Sal, a Lean 4-based framework using Small Proof-Oriented Tests (SPOTs) to close the gap.

Pranav Ramesh and Vimala Soundarapandian co-authored the paper "Sal: Multi-modal Verification of Replicated Data Types", presented at the same workshop.

FIDES at AsiaCCS 2026

Sai Venkata Krishnan and collaborators published "FIDES: End-to-end Compartments for Mixed-language Systems" at AsiaCCS 2026. FIDES uses hardware-assisted compartments on the Shakti RISC-V platform to isolate components in mixed-language programs, limiting the blast radius of bugs across language boundaries.
